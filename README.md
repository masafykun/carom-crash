# CAROM CRASH

A 3D bank-shot slingshot demolition: kiss the neon panels to ricochet your ball around cover and into the fortress — every bank multiplies the destruction.

**▶ Play in browser:** https://masafykun.github.io/carom-crash/

![screenshot](screenshot.png)

## About
A small game built with Unity (6000.0.77f1). The C# source is under `src/`.
This repository also hosts a WebGL build, playable directly in the browser via GitHub Pages.
